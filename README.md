# Realtime Whiteboard 🎨

A collaborative drawing app where multiple users can draw on the same canvas in real time.

## Live Demo

https://realtime-whiteboard-production-155f.up.railway.app

## Features

- Real-time drawing sync between multiple users
- Color picker
- Adjustable brush size
- Clear canvas syncs to all connected users

## Tech Stack

- Frontend: HTML, CSS, Vanilla JavaScript, Canvas API
- Backend: Node.js, Express, Socket.io
- Deployed: Railway

## How to run locally

Clone the repo, navigate to the backend folder, run `npm install` then `node server.js`, and open `http://localhost:3001` in your browser.
